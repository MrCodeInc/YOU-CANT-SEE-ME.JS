YOUCANTSEEME.js

A simple javascript bookmarklet to hide almost all images on a webpage.

Download Download View YOUCANTSEEME.js on GitHub

HOW TO INSTALL.

Copy the following code: javascript:(function(){var%20imgs=document.getElementsByTagName("img");for(var%20i=0;i<imgs.length;i++)imgs[i].style.visibility="hidden"}());(function()%7Balert%20(%22YOU%20CANT%20SEE%20ME%22)%7D)() and then make a new bookmark. Paste in the code as the destination of your new bookmark and save it. Now just click the bookmark and it should work.

HOW TO SHOW IMAGES AGAIN.

To show images again refresh the page that the images have been hidden on and images should be back.
